# 📚 Multi-Subject English-Urdu Dictionary

A comprehensive, production-ready dictionary web application supporting multiple subjects with bilingual search (English-Urdu), user authentication (Admin/Member roles), and bulk upload capabilities.

## 🌟 Features

### 👑 Admin Features
- **Dashboard Analytics** - View statistics, charts, and popular words
- **Subject Management** - Create, edit, delete subjects with custom icons and colors
- **Word Management** - Add, edit, delete words in any subject
- **Bulk Upload** - Upload thousands of words via Excel/CSV files
- **User Management** - Create and manage member accounts
- **Global Search** - Search across all subjects and words

### 👥 Member Features
- **Browse Subjects** - View all available subjects with word counts
- **Subject-wise Viewing** - Explore words within specific subjects with pagination
- **Bidirectional Search** - Search English to Urdu and Urdu to English
- **Word Details** - View Urdu meanings, English meanings, examples, synonyms, antonyms
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
### 🔐 Security Features
- JWT-based authentication
- Role-based access control (Admin/Member)
- BCrypt password encoding
- Secure API endpoints

## 🛠️ Technology Stack

- **JWT** - Token-based authentication
- **Apache POI** - Excel file processing

- ### Frontend
